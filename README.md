# Netflix Content Analysis

## Project Overview

This project analyzes Netflix's content catalog to uncover trends in:

- Movies vs TV Shows
- Country-wise content distribution
- Genre popularity
- Release year trends
- Movie duration analysis

The project uses Python, Pandas, Matplotlib, and Seaborn for data analysis and visualization.

---

## Business Question

What trends exist in Netflix content?

---

## Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- VS Code

---

## Dataset

Netflix Movies and TV Shows Dataset

Source:
https://www.kaggle.com/datasets/shivamb/netflix-shows

---

## Analyses Performed

### 1. Movies vs TV Shows
Compared the number of Movies and TV Shows available on Netflix.

### 2. Country-wise Content
Identified top countries producing Netflix content.

### 3. Genre Trends
Analyzed the most common genres.

### 4. Release Year Trends
Studied content growth over time.

### 5. Duration Analysis
Examined movie length distribution.

### 6. Heatmap Analysis
Visualized release trends by content type.

---

## Key Insights

- Movies dominate Netflix's catalog.
- The United States contributes the highest number of titles.
- International Movies and Dramas are highly popular.
- Netflix content production increased significantly after 2010.
- Most movies range between 80 and 120 minutes.
